# 전역 변수의 문제점

## 변수의 생명주기

전역 변수 선언문 -> 런타임 이전에 먼저 실행된다

지역 변수 -> 함수의 생명 주기와 일치한다

일반적으로 함수가 종료되면 생성한 스코프도 소멸된다

> 하지만 누군가가 스코프를 참조하고 있다면 스코프는 생존한다 ==> 클로저

```
var x = 'global'

function foo () {
    console.log(x);
    var x = 'local';
}
foo(); // undefined
console.log(x); //global
```

- 호이스팅은 스코프 단위로 동작한다

## 전역 변수의 생명주기

var 키워드로 선언한 변수의 생명 주기 == 전역 객체의 생명 주기
